# ISO2USB  

    ISO2USB v0.1beta;  
    Create a bootable USB drive.  

## Usage:  
    ./iso2usb.sh  [ISO]... [DISK]...  
    ./iso2usb.sh  [INPUT]. [OUTPUT].  
  
## Example:  
    ./iso2usb.sh  '/user/directory/image-file-1.0-amd64.iso' '/dev/disk'  
    ./iso2usb.sh  '/user/directory/image-file-1.0-amd64.iso' '/dev/sdb6'
